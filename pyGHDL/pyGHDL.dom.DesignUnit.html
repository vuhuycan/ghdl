pyGHDL.dom.DesignUnit¶

This module contains all DOM classes for VHDL’s design units (entity, architecture, package, package body, context and configuration.

Classes

  • Entity: ModelEntity is a base class for all classes in the VHDL language model,

  • Architecture: ModelEntity is a base class for all classes in the VHDL language model,

  • Package: ModelEntity is a base class for all classes in the VHDL language model,

  • PackageBody: ModelEntity is a base class for all classes in the VHDL language model,

  • Context: ModelEntity is a base class for all classes in the VHDL language model,

  • Configuration: ModelEntity is a base class for all classes in the VHDL language model,
